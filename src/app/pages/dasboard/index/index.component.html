Focus Mode Features

When activated, Focus Mode will:

• Hide the navigation history (top-left)

• Hide the user profile (top-center)

• Hide the main navigation items (bottom-left)

• Expand content to full width

• Keep only the Focus Mode toggle visible for easy exit

How to Exit Focus Mode

You can exit Focus Mode in two ways:

1. Click the Focus Mode toggle button again (it will have a blue gradient)

2. Press the ESC key on your keyboard
